What is El Digito?

El Digito is an AI agent, which creates random digits which look like human-handwritten digits. El Digito works on the principle of GAN, a generative model.

What does the Custom Slider Input do?

The model needs a gaussian sample array. Each time you press Generate, a random gaussian sample array is generated of mean equaling the Slider Input. Each Input Number represents a visually similar cluster of digits. Also, closer numbers (for ex: 6 and 8 would generate similar digts only)
